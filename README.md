<h1>System resource monitoring tool</h1>

#<h2>Build</h2>
1. sudo apt-get install -y libboost-all-dev
2. cd Server
3. ./build.sh


#<h2>Help</h2>
: ./resource_tool -h


#<h2>Tool Usage Guide</h2>
1. show only local <br>
: ./resource_tool -l -n {network interface}
2. run server <br>
: ./resource_tool -s -n {network interface}
